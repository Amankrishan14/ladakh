# SBMU Banner Assets

This directory should contain the following image files for the SBMU banner component:

## Required Files:

1. **lion.png** - Large 3D lion mascot image (displayed at bottom center of banner)
   - Recommended size: ~300px width
   - Should be transparent background PNG

2. **sbmu-header.png** - Header logo with smaller lion and Gujarati text (displayed on right side)
   - Recommended size: ~140px width
   - Should be transparent background PNG

## File Structure:
```
src/ui/assets/sbmu/
├── lion.png
├── sbmu-header.png
└── README.md (this file)
```

## Note:
If these files are missing, the component will fail to build. Please add the actual image files to this directory.
